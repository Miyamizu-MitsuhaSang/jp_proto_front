# JP Proto Front

## 变更说明

v0.1.1 (2026-01-24)
- 录音页新增朗读文本复制提示弹窗，并支持顶部提示展示。
- 录音页学习时长使用自定义上下箭头，禁用 0 时减号按钮。
- 录音页禁用文件上传，只允许直接录音提交。
- 管理页整体优化：表格可横向滚动、阻止触控板返回手势、Overall 分数按规则着色。
- 管理页表头“学习年限”改为“学习时长”。
- 管理入口点击先发 ping 校验，未授权才弹出密码框。

v0.1.0 (2026-01-23)
- 管理端朗读文本删除改为确认弹窗（无蒙层、居中显示，尺寸随窗口自适应，支持滚动）。
- 管理端“刷新列表”保持当前页，不再回到第一页。
- 新增文本输入框为空时保持单行高度，输入后自动扩展。
- 管理端记录列表刷新保持当前页。
